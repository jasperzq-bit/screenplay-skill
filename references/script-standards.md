# 剧本写作标准

台词、场景描述、运镜备注三个模块的写作规范。写剧本正文前必读。

---

## 一、台词写作标准

### 1.1 真人呼吸感：什么是，什么不是

**真人说话的本质特征：**
- 不完整：说到一半停住，被打断，想起别的事
- 不合逻辑：情绪来了说错话，或者说反话
- 有节奏：快慢混合，有的地方堆词，有的地方一字一顿
- 有个人习惯：惯用词、口头禅、特定句式

**AI腔的本质特征（写作时禁止出现）：**
- 每句话逻辑完整，有起承转合
- 角色用台词解释自己的感受和动机
- 说谢谢/对不起/我理解/我明白这类整洁回应
- 一段对话像在填满剧情空白，而不是两个人真的在说话

### 1.2 台词密度原则（短剧专用）

短剧里每句台词的存在都要有代价。问自己：这句话如果删掉，观众会损失什么信息或情绪？损失了才留，没损失就删。

**台词的四种功能（任何一句台词至少完成其中一种）：**
1. 推进情节（提供信息/揭示秘密/触发行动）
2. 推进情绪（让观众的心往某个方向移动）
3. 刻画角色（这只有这个人能这么说）
4. 埋钩子（让观众想知道接下来发生什么）

### 1.3 台词格式规范

```
人物名（情绪/动作说明）
台词内容

人物名
台词内容
```

括号里的说明：
- 只写对表演/生图有用的信息，不写多余说明
- 用动词或形容词，不用"他感到很难过地说"
- 示例（对）：`（轻声，没有回头）` `（眼神从他脸上移开）` `（停了很久）`
- 示例（错）：`（用悲伤而又复杂的心情说道）`

### 1.4 停顿和打断的写法

**停顿**：用（停）或（停顿）或（…）内嵌在台词里，不要独占一行

```
林昊
就这样。（停）你满意了？
```

**打断**：用——结束被打断的话

```
江晨
昊哥你听我说，这件事其实——

林昊
不用。
```

**说到一半停住**：用……或（顿了一下）

```
沈悦
我不是……（顿了一下）算了。
```

### 1.5 各角色台词检查方法

写完一场戏后，把所有台词的人物名遮住，只读台词。

- 能分辨出哪句是哪个人说的 → 合格
- 分辨不出 → 重写，找出是哪个角色失去了自己的声音

## 二、场景描述写作标准

### 2.1 场景描述的三个层次

**第一层（必须有）：空间定位**
- 地点（具体，不能只写"餐厅"，要写"一家只有四张桌子的老式面馆，午后没什么客人"）
- 时间（白天/夜晚/特定时刻，以及光线状态）
- 内/外景

**第二层（必须有）：氛围营造**
- 主光源和光线质感（冷光/暖光/逆光/阴天散射光）
- 2–3个关键陈设（选最能体现氛围的，不列清单）
- 整体氛围关键词（1–2个词，如"破败的安静" / "浮夸的空洞"）

**第三层（选填，重要场景必须有）：情绪锚点**
- 这个空间里，什么东西最能钩住观众的视线？
- 什么东西是后续剧情的道具线索？（提前在场景描述里埋）

### 2.2 场景描述的字数要求

- 功能性场景（过场，1–2句台词）：2–3句，定位清楚即可
- 普通对话场景：4–6句，有空间感和氛围
- 重要情绪场景（高潮/转折）：6–10句，细节必须支撑情绪

### 2.3 场景描述格式

内嵌在场景标题下方，用括号包裹，斜体（文档里用斜体，代码里用括号）：

```
【场景01·天台·夜·外景】

（宽阔的天台，城市的灯火在远处低低连成一片，
  风从没有遮挡的方向吹来。
  地面的防水层已经开裂，有几处积水，
  反射着对面楼的冷光。
  整个空间给人一种高处的孤立感，
  像是被城市遗忘的边缘。
  AI复杂度：中等。）
```

### 2.4 禁止在场景描述里出现的内容

- 形容词堆砌（"奢华的、华丽的、精致的"——选一个体现在一个具体细节上）
- 情绪直接说明（不写"这是一个令人压抑的房间"，写让人感到压抑的具体陈设）
- 超出画面范围的信息（旁白式的背景解释放到梗概里，不放场景描述）

## 三、运镜备注写作标准

### 3.1 运镜备注的作用

运镜备注不是给观众看的，是给导演/生视频AI看的。它说明：
- 这场戏从哪个景别开始，往哪里推进
- 关键情绪节点用什么镜头处理
- 镜头的移动逻辑

### 3.2 景别选择原则

| 景别 | 用途 |
|---|---|
| 宽全景 / 大全景 | 建立空间，交代环境，人物在空间里的位置关系 |
| 全景 | 人物全身，展示肢体动作和站位 |
| 中景 | 腰部以上，日常对话，人物关系的空间距离 |
| 近景 | 胸部以上，情感靠近，重要台词 |
| 特写 | 脸/手/道具，情绪最高点，关键细节 |
| 极端特写 | 眼睛/嘴角/某个细节，爆发时刻 |

**基本规律：**
- 情绪升温 → 景别越来越小（全景→中景→近景→特写）
- 空间交代 → 全景
- 人物关系 → 中景对切
- 情绪高潮 → 特写/极端特写定格

### 3.3 运镜备注格式

内嵌在场景描述下方，单独一行，用【】标注：

```
【运镜：宽全景建立天台空间→缓推至林昊背影中景→切至江晨出现时过肩拍→
  对话来回中近景切换→"十八年"这句话后停顿，特写：林昊手里的U盘。】
```

简洁即可，不需要写成分镜脚本。核心是：起点景别 + 推进方式 + 情绪高点的镜头处理。

### 3.4 常用运镜描述词汇

- **推镜**：摄影机向主体靠近，产生压迫感或亲近感
- **拉镜**：摄影机远离主体，产生孤立感或失落感
- **跟拍**：跟随人物移动，临场感强
- **固定机位**：稳定，强调人物在静止空间里的张力
- **过肩拍**：A过肩看B，强调两人关系和距离
- **定格**：在关键情绪节点停止运动，让观众多停留一秒

## 四、动作描述写作标准

### 4.1 动作描述的原则

- 一个动作一句话，不叠加
- 用动词不用形容词（"他把日历放回去"，不是"他小心翼翼地、轻柔地把日历放了回去"）
- 动作要有意义：能反映心理状态、推进情节，或提供视觉信息
- 微表情和细节动作比大动作更有戏剧性

### 4.2 微表情的写法

不要直接写情绪（"他很愤怒"），写身体反应：

- 错误：他愤怒地看着江晨
- 正确：他看着江晨，拳头在裤缝处慢慢收紧

- 错误：她很害怕，紧张地站在那里
- 正确：她的手指在包带上收紧了，没有动

### 4.3 沉默场景的写法

沉默是台词，不是空白。沉默里有动作、有眼神、有空气的质量。

```
两个人都没有说话。

林昊把手机放回口袋。
他没有看江晨，看的是远处的灯火。

江晨的笑容还挂着，但是维持了太长时间，那个笑就变成了别的东西。
```

## 五、短剧专项节奏标准

### 5.1 短剧节奏铁律

短视频平台用户0.5秒没抓住就划走。以下是不可妥协的节奏标准：

**时间节点要求：**

- **第1帧（0秒）**：必须是全集情绪最高点之一，或最强悬念画面。
  观众看到第一帧，必须产生「这是什么」或「接下来怎样」的问题。
  不允许用任何「建立背景」「交代环境」的镜头作为第一帧。

- **前3秒**：必须完成一个情绪冲击或信息投放。
  冲击类型：
  视觉冲击（激烈动作/强烈反差画面）/
  台词冲击（一句让人无法忽视的话）/
  悬念冲击（一个明显不正常的画面细节）

- **前10秒**：必须让观众感受到「有事要发生了」。
  不需要解释来龙去脉，但核心冲突必须已经在画面或台词里出现。

- **第一句台词**：必须是信息炸弹或情绪炸弹。
  信息炸弹示例：「你知道我查了多少年吗？」
  情绪炸弹示例：「谢谢你的股份。」
  禁止用废话开场：「今天天气真好」「你来了」「吃了吗」「我们谈谈」

**每集开场禁止清单（出现即重写，不做解释）：**

- 主角一个人沉默站着/坐着/看窗外超过3秒
- 环境描写超过3秒才出现人物声音或对话
- 第一场戏只有一个人物，没有任何其他人介入或声音触发
- 用旁白、画外音或字幕交代背景信息作为开场
- 开场情绪是「平静」「沉思」「回忆」「独处」
- 第一个镜头是风景、建筑或纯空镜（无人物、无声音）

**合格开场的三种模式（任选其一）：**

模式A·冲突爆发型：
直接从最激烈的冲突场面开始，观众被扔进最高点，
故事再往前或往后解释来龙去脉。
示例：第一帧是林煜辰被推下天台的瞬间，
第一句台词是顾明远的笑声或一句冷静的台词。

模式B·信息炸弹型：
第一句台词或画外音揭示核心秘密或最大矛盾，
让观众带着强烈问题往后看。
示例：第一帧是林煜辰看着捐赠书，
画外音：「顾明远，你以为这就结束了。」

模式C·强反差型：
画面和声音/台词形成强烈反差，制造悬念。
示例：第一帧是顾明远在宴会上满面春风，
画外音是林煜辰冷静的声音：「你偷走的，我一件一件拿回来。」

### 5.1.1 首帧描述框格式

每集剧本正文开始前，主 agent 必须填写以下描述框，
填完后必须交给审核子 agent 审核，禁止自查：

【首帧描述框·第X集】
第一帧画面：（一句话，描述观众看到的第一个具体画面，
             必须有人物或强烈视觉元素）
第一个声音或台词：（逐字写出，不能写「某人说了一句话」）
前3秒完成的情绪任务：（观众在3秒内产生了什么问题或情绪）
使用的开场模式：从A/B/C中选一个主模式，
如果同时满足第二种模式可标注为副模式。
格式示例：「主：A冲突爆发型，副：B信息炸弹型」
不允许三种全选，不允许不选。
禁止清单逐条核查：
- 主角独处沉默超过3秒：有 / 无
- 环境描写超过3秒无人声：有 / 无
- 第一场戏只有一个人物：有 / 无
- 用旁白字幕交代背景：有 / 无
- 开场情绪是平静/沉思/独处：有 / 无
- 第一个镜头是纯空镜无人物：有 / 无

当开场同时符合多种模式时，选最主要的那个作为主模式标注，
第二种作为副模式标注，不需要强行只选一个。

### 5.2 情绪节点密度

每集剧本写完后，标出所有情绪高点（用🔥标记）。

检查：
- 每隔30–45秒有一个🔥
- 相邻两个🔥之间有情绪缓冲（不能每秒都是高潮，要有呼吸）
- 全集最大的🔥在集中后段（2/3处之后）

### 5.3 集尾钩子

集尾最后10秒，必须留下一个让观众"不看下集睡不着"的悬念。

钩子类型：
- **行动悬念**：主角做了一个动作，但不知道结果（打出了那个电话/走进了那扇门）
- **信息悬念**：揭示了一条新信息，但更大的问题随之而来
- **情绪悬念**：某个角色的反应出乎意料，观众需要下一集解释
- **威胁悬念**：反派已经发现了什么/布下了某个局，主角还不知道

❌ 绝对不能温和收口（"主角回到家，睡着了"这类）
❌ 绝对不能道德总结（"他/她明白了一个道理……"）
