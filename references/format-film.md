# 单体短片 · 完整输出格式规范

适用范围：独立短片、悬疑短片、文艺短片、广告片、品牌故事片。
时长通常3–15分钟，不分集，一次性完整呈现。

---

## 输出模块顺序（固定，不可省略）

```
01. 剧本基础信息表
02. 故事梗概
03. 人物设定（每个人物完整五板块）
04. 人物关系图谱
05. 段落结构大纲（按幕/段划分）
06. 完整剧本（按场次展开）
```

与短剧格式的核心区别：
- 无"核心冲突线 + 情绪曲线"模块（改为段落结构大纲中体现）
- 无"分集"概念（改为"幕/段"）
- 结尾可以是收口（短片允许闭环结局），但要有情感余韵

---

## 01. 剧本基础信息表

| 字段 | 内容 |
|---|---|
| 片名 | |
| 类型 | 如：悬疑短片 / 文艺短片 / 品牌故事片 / 恐怖短片 |
| 总时长 | |
| 目标受众 | |
| 核心卖点 | 一句话，让人想看的那句话 |
| 情绪定位 | 如：悬疑惊悚 / 温情治愈 / 反转爽感 / 压抑沉重 |
| 平台适配 | 如：电影节投递 / 社交媒体 / 品牌发布会 / 竖屏移动端 |
| AI生成备注 | 场景复杂度要求，人物数量上限 |

---

## 02. 故事梗概

**一句话梗概**：主角 + 核心困境 + 情绪温度，20–30字。

**完整梗概**：400–800字（短片比短剧更精炼）。包含：
- 故事发生的时间/空间背景
- 主角的起点状态
- 核心事件如何触发
- 情绪/悬念如何推进
- 结局或留白方式

短片梗概更注重情绪氛围和主题，比短剧梗概更有文学性。

---

## 02.5 核心事件真相表

**必须在人物设定之前完成。未完成此表，不得进入人物设定环节。**

这张表的作用是把故事里所有「幕后发生的关键事件」显式拆解，
强制验证每一个事件的执行者、动机、信息掌握程度和知情范围。

对于悬疑短片，此表是创作前的核心功课——结局能否自圆其说，
完全取决于这张表在第一个镜头写出来之前是否填严。

对于含广告植入的短片，品牌出现的每一个场景也需要在此表里
作为独立条目填写并通过验证（详见本文件末尾「广告植入规范」）。

**「关键事件」的定义：**
- 推动故事主线的幕后行为（如：谋划、隐瞒、背叛、牺牲）
- 在剧情中已发生、但观众稍后才知道的事件
- 悬疑短片中：所有指向结局反转的隐藏信息和行为
- 含植入广告的短片中：品牌/产品出现的每一个场景

**填写格式（每个关键事件一行，不可合并）：**

| # | 事件描述（具体，不可模糊） | 执行者 | 执行者动机（必须具体到个人处境） | 执行者当时掌握的信息 | 知情者 / 不知情者 |
|---|---|---|---|---|---|
| 01 | | | | | |
| 02 | | | | | |

**每填完一行，必须回答以下三个问题。全部回答「是」才可继续：**

问题一：这个执行者有合理动机做这件事吗？
动机必须具体到这个人的处境、欲望或恐惧，不能写抽象描述。

问题二：这个执行者当时掌握的信息，足以让他做出这个判断和决定吗？

问题三：这个事件和其他已填事件之间，有没有时间线或因果链的矛盾？

**悬疑短片专项检查（非悬疑类型可跳过）：**

填完整张表后，额外回答以下问题：

- 结局反转所依赖的每一条关键信息，在故事前段都有对应的铺垫吗？
  （反转必须能被观众回溯验证，不能是无中生有）
- 凶手/真相执行者的动机，在结局揭露时观众会觉得「原来如此」
  还是「这不合理」？
- 所有「误导观众」的信息，在故事世界内部是否仍然自洽？
  （可以欺骗观众，但不能欺骗故事本身的逻辑）

---

## 03. 人物设定

与短剧人物设定规范相同，读 `references/character-standards.md`。

短片特殊要求：
- 人物数量控制在2–4人（人物越少，情感越深）
- 每个人物的内心动机必须写清楚，因为短片靠内心而非外部冲突驱动
- 次要人物可以简化，但主角必须完整五板块

---

## 04. 人物关系图谱

格式同短剧，但短片人物关系通常更简单，重点标注关系的情感层次（表面关系 vs 内里真相）。

---

## 05. 段落结构大纲

短片按"幕"或"段落"划分，不按集数。

| 幕/段 | 时长 | 核心内容 | 情绪/功能 |
|---|---|---|---|
| 开场 | X分钟 | 建立角色+世界+核心问题 | 吸引/建立期待 |
| 发展 | X分钟 | 冲突升级/情感推进 | 主情绪 |
| 高潮 | X分钟 | 最大情绪爆发点/反转 | 爆发/震撼 |
| 结尾 | X分钟 | 收口或留白 | 余韵/反思 |

根据实际时长和类型灵活调整幕数（3–5幕）。

---

## 06. 完整剧本

### 场景块格式（与短剧相同）：

```
【场景编号·地点·时间·内/外景】

（场景视觉描述：空间大小、光线、主要陈设、氛围关键词。
  必须具体到可以生图的程度。AI复杂度：简单/中等/复杂。）

【运镜备注：开场景别→推进方式→情绪节点的镜头处理】

人物动作描述

人物名（情绪/动作说明）
台词内容
```

### 短片节奏要求（与短剧的区别）：

- **开场**：不必"前15秒强钩子"，但必须在开场确立一个让人想看下去的核心问题或情绪氛围
- **节奏**：比短剧更缓，允许呼吸感更长的停顿，情绪可以慢慢渗透
- **结尾**：可以是闭合结局，也可以是开放性留白，但必须有情感余韵——观众看完后要有一个停留感，而不是"就这样结束了？"
- **台词密度**：比短剧更低，允许更多沉默和动作替代台词

### 悬疑/惊悚短片额外要求：

- 信息投放节奏：前1/3建立"正常世界"，中段开始放疑点，后1/3加速揭露
- 每次疑点的出现必须有具体视觉锚点（某个道具/某句台词/某个细节）
- 结局反转必须有前期铺垫，不能无中生有
- 氛围描述要更细，包含声音设计提示（静音/环境音/某个特定声响的出现）

### 品牌短片额外要求：

- 产品/品牌的出现方式必须自然，不能硬植入
- 情感钩子先于品牌信息，观众先被故事打动，再接受品牌
- 结尾的品牌露出必须与故事情绪形成承接，不能突兀

## 广告植入规范

适用场景：
- 以广告为核心目的的独立品牌短片
- 在短剧单集或短片中植入品牌/产品

两种场景处理逻辑不同，分别说明。

---

### A. 独立广告短片

整个故事为品牌服务，品牌是创作起点而非终点。

**创作顺序（不可颠倒）：**
1. 先明确品牌核心传达的情感或价值观
   注意：是情感，不是产品功能。「更快」「更强」不是情感，
   「不被辜负」「终于被看见」才是情感。
2. 围绕这个情感找一个能承载它的故事内核
3. 写完故事后，检查品牌出现的每一个时机是否与故事情绪自然承接

**品牌信息表（必须在故事梗概之前填写）：**

| 字段 | 内容 |
|---|---|
| 品牌名 | |
| 产品或服务类型 | |
| 核心传达情感 | 只能选一个，不能多。多个情感等于没有情感 |
| 目标受众情感痛点 | 这个受众在生活里最容易被什么场景触动？ |
| 品牌出现方式 | 道具 / 台词 / 场景 / 行为（见下方分级说明） |
| 品牌出现时机 | 情绪高点之后 / 情绪转折处 / 结尾（三选一） |
| 品牌禁区 | 什么场景或情绪下，品牌绝对不能出现？ |

品牌信息表填完后，将其纳入「核心事件真相表」，
把每一次品牌出现作为独立条目填写并完成验证，才可以开始写故事。

---

### B. 短剧或短片中的广告植入

品牌需要融入一个已有故事，不能破坏故事的逻辑和情绪节奏。

**核心原则：故事逻辑和品牌逻辑必须同时成立。**

植入失败的根本原因永远只有一个：品牌的出现在故事里没有动机支撑，
观众感受到了「切换」——从故事世界切换到了广告世界。
这个切换一旦发生，故事情绪断掉，品牌形象也受损。

**植入方式分级（从最自然到最生硬）：**

| 级别 | 方式 | 说明 | 适用条件 |
|---|---|---|---|
| 一级（最自然） | 行为植入 | 人物使用产品是故事情境的必然需要，不需要特别展示品牌 | 人物性格/职业/生活习惯与品牌高度契合，植入后感觉「本来就应该在这里」 |
| 二级 | 道具植入 | 产品作为场景道具出现，有视觉存在感但不打断叙事 | 产品在视觉上与场景氛围匹配，出现时机在情绪平稳期 |
| 三级 | 场景植入 | 故事发生地点是品牌相关的空间（门店/品牌环境） | 地点对故事有叙事意义，不只是为了展示品牌 |
| 四级 | 台词植入 | 人物主动提及品牌名或产品特点 | 仅在有充分人物动机时使用。人物提及品牌，必须是这个人在这个时刻说这句话是自然的 |
| 五级（尽量避免） | 强制植入 | 旁白、字幕或非叙事性画面强行插入品牌信息 | 合同硬性要求且无法回避时才用，必须控制在5秒以内，放在集尾或片尾 |

**植入时机选择原则：**

适合植入的时机：
- 情绪刚刚完成一个高点，观众处于短暂的情感满足状态
- 故事自然过渡到下一场景的间隙
- 人物做出一个决定之后，正在行动的过程中（行为自然带出产品）

禁止植入的时机：
- 情绪高潮正在进行中（强行切断观众的情感投入）
- 台词密度最高的对话场景中间（破坏节奏，观众会感到突兀）
- 悬念刚刚抛出、观众正在紧绷等待的时候
- 任何负面情绪场景中（悲剧/激烈冲突/人物痛苦）——品牌不能和负面情绪绑定

**植入频次控制：**

| 内容类型 | 最多植入次数 | 间隔要求 |
|---|---|---|
| 单体短片（5分钟以内） | 2次 | 两次植入之间至少间隔1分钟叙事内容 |
| 短剧单集（1.5–3分钟） | 1次 | 单集内不出现两次 |
| 短剧全季 | 每3集不超过2次 | 不在相邻两集的同一叙事位置出现 |

超出以上频次的植入，无论客户如何要求，都必须在剧本备注里标注
「此频次存在影响内容质量的风险」，由创作方决策是否接受。

**植入动机验证（写剧本前必须完成）：**

在「核心事件真相表」里，为每一个植入场景新增一行，填写以下内容：

| # | 植入场景描述 | 使用/提及品牌的人物 | 人物动机（为什么这个人在这个时刻使用这个产品） | 故事逻辑自洽？ | 品牌逻辑自洽？ |
|---|---|---|---|---|---|

两项都是「是」，才可以保留这个植入场景。
任何一项是「否」，二选一：修改故事情境让植入动机成立，或取消这个植入点。

**双重逻辑验证清单（含植入内容的剧本写完后执行）：**

故事逻辑层面：
- [ ] 品牌出现的场景，在故事世界里有合理存在的理由？
- [ ] 使用或提及品牌的人物，有符合其性格和处境的动机？
- [ ] 品牌出现没有打断当集或当片的情绪节奏？
- [ ] 植入场景删掉之后，故事依然完整成立？
      （真正自然的植入，删掉后故事会有缺失感；如果删掉反而更流畅，说明植入是硬的）

品牌逻辑层面：
- [ ] 品牌出现时的情绪氛围与品牌形象相符，没有负面情绪绑定？
- [ ] 品牌传达的核心情感在这个场景里得到了体现，不只是露出了品牌名？
- [ ] 植入方式和频次符合「植入方式分级」和「频次控制」的规范？
