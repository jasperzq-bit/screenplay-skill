# 生图 Prompt 转译规范

本文件定义如何将 screenplay skill 输出的人物形象描述和固化场景描述，
转译为可直接输入 LibTV（Seedream 模型）的生图 prompt。

转译是一次语言切换：从「给人读的叙事描述」变成「给模型看的视觉指令」。
两套语言规则完全不同，不能混用。

---

## 一、何时执行转译

以下两种情况，在输出完成后自动追加生图 prompt：

1. 完成人物设定的「C. 详细形象描述」之后
2. 完成固化场景库的每一个场景描述之后

格式：在原描述下方追加一个独立区块，标题为：
「🎨 LibTV 生图 Prompt」

---

## 二、人物生图 Prompt 转译规则

### 必须保留的内容（逐条提取）

| 原描述内容 | 转译方式 |
|---|---|
| 人种、性别、年龄 | 直接写，必须是第一句 |
| 身形 | 用描述词，禁止数字（178cm→高挑身形） |
| 脸型 | 直接写 |
| 五官逐条 | 眉型、眼型、鼻型、唇型分别写 |
| 肤色 | 保留，去掉比喻部分 |
| 气质关键词 | 最多2个，必须是视觉可感知的词 |
| 着装颜色+款式 | 保留，去掉象征意义的解释 |

### 必须删除的内容

- 所有性格描述（克制、冷、精准……）
- 所有心理状态（他不信任任何人……）
- 所有比喻性语言（像压了很多年的木头……）
- 所有行为习惯描述（说话前停顿半秒……）
- 所有数字身高（统一改为描述词）
- 「情绪变化下的外貌」整节（生图是静态的，不需要）
- 「标志性细节」里的行为部分，只保留外形部分

### 必须追加的固定技术词（每张人物图都加）


头部完整可见，全身照，脚部完整，白色背景，
正面站姿，写实风格，电影质感，
高清摄影，自然光，无水印。


### 风格锁定词（根据角色类型选一个）

| 角色类型 | 风格锁定词 |
|---|---|
| 中国男性角色 | 中国男演员气质，非韩流风格 |
| 中国女性角色 | 中国女演员气质，东方面孔 |
| 韩系风格（用户指定） | 韩国男演员气质，韩流风格 |
| 西方角色 | 欧美男/女演员气质，西方面孔 |
| 古装角色 | 中国古装，汉服，古典东方美学 |

### 人物生图 Prompt 固定格式


[人种+性别+年龄]，[身形描述]，[脸型]，
[五官：眉/眼/鼻/唇]，[肤色]，[下颌线/轮廓]。
[着装：颜色+款式+材质]，[配饰（如有）]。
[气质关键词，最多2个]。
[风格锁定词]。
头部完整可见，全身照，脚部完整，白色背景，
正面站姿，写实风格，电影质感，高清摄影，自然光，无水印。


### 转译示例

原描述（screenplay skill 输出）：
> 约178cm，身形偏瘦，肩背却是直的，像压了很多年终于能抬起来的木头。
> 脸型偏长，轮廓硬，眉骨高，眼窝略深，眼神有一种习惯性的冷。
> 鼻梁挺直，嘴唇偏薄。肤色偏白，但不是养尊处优的白。
> 着装偏爱深色系，款式简洁、质地良好但不张扬。
> 他穿衣服是在给自己造一层壳。

转译后 LibTV Prompt：

中国男性，26岁，高挑身形修长偏瘦，肩膀宽而挺直。
脸型偏长，轮廓硬朗，眉骨高耸，眼窝略深，单眼皮，
眼神冷静锐利。鼻梁高挺，嘴唇偏薄。
肤色白皙有质感，下颌线清晰有棱角。
黑色高领毛衣，深灰色西装外套，简洁无装饰。
冷峻气质，电影男主风格。
中国男演员气质，非韩流风格。
头部完整可见，全身照，脚部完整，白色背景，
正面站姿，写实风格，电影质感，高清摄影，自然光，无水印。

## 三、场景生图 Prompt 转译规则

### 必须保留的内容

| 原描述内容 | 转译方式 |
|---|---|
| 地点类型 | 保留，具体化（"室内"→"现代公寓客厅"） |
| 时间/光线 | 保留，转为光线描述词 |
| 主要陈设 | 最多3个关键道具，不列清单 |
| 氛围关键词 | 保留，最多2个 |
| AI复杂度标注 | 保留，直接写入 |

### 必须删除的内容

- 所有情绪解释（"这个空间让人感到压抑"→删除解释，保留陈设）
- 人物动作描述（场景图不含人物，或单独说明）
- 运镜备注（运镜是视频阶段的事）

### 必须追加的固定技术词


无人物，空镜，写实风格，电影质感，
高清摄影，[光线类型]，无水印。


### 场景生图 Prompt 固定格式

[地点类型，室内/室外]，[时间：白天/夜晚/黄昏]，
[主光源描述]，[关键陈设1]，[关键陈设2]，[关键陈设3（可选）]。
[氛围关键词，最多2个]。
[AI复杂度：简单/中等/复杂]。
无人物，空镜，写实风格，电影质感，
高清摄影，室内人工光，无水印。


### 转译示例

原描述（screenplay skill 输出）：
> 沈国强书房，约四十平米的私人书房，深棕色为主。
> 书桌是一张老式红木办公桌，桌面上放着一个旧相框——
> 照片背面朝外放着，看不到照片。书架上排满管理学书籍，
> 没有一本看起来被翻过。灯光：桌面台灯白光加天花板暖光，
> 冷暖对冲。氛围关键词：压抑的仪式感。AI复杂度：简单。

转译后 LibTV Prompt：
```
现代豪宅私人书房，室内，夜晚，
桌面台灯冷白光与天花板暖光混合，冷暖对冲。
老式红木办公桌，背面朝外的旧相框，
整齐但从未翻阅过的书架。
压抑感，仪式感，深棕色调。
AI复杂度：简单。
无人物，空镜，写实风格，电影质感，
高清摄影，室内人工光，无水印。

## 四、三视图生成说明

人物参考图生成完成后，进入 LibTV「角色三视图」节点时，
在连线输入的同时，追加以下补充说明：


基于参考图生成角色三视图：正面、左侧面、背面。
保持与参考图完全一致的：面部特征、发型、着装、体型。
白色背景，全身照，头部和脚部完整可见，无水印。

---

## 五、转译执行检查清单

每次生成 prompt 后，过一遍以下检查：

- [ ] 有没有出现数字身高？→ 改为描述词
- [ ] 有没有性格/心理描述？→ 全部删除
- [ ] 有没有比喻性语言？→ 全部删除
- [ ] 人种风格词加了吗？→ 没有则补上
- [ ] 固定技术词加了吗？→ 没有则补上
- [ ] 场景图说明「无人物，空镜」了吗？→ 没有则补上
