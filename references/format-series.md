# 多集竖屏短剧 · 完整输出格式规范

## 输出模块顺序（固定，不可省略任何一项）

```
01. 剧本基础信息表
02. 故事梗概
03. 核心冲突线 + 情绪曲线设计
04. 人物设定（每个人物完整五板块）
05. 人物关系图谱
06. 分集大纲
07. 分集剧本
```

---

## 01. 剧本基础信息表

一张表，开门见山，固定字段，全部必填：

| 字段 | 内容 |
|---|---|
| 剧名 | |
| 题材 | 如：重生爽剧 / 豪门虐恋 / 替身打脸 / 都市逆袭 |
| 总集数 | |
| 单集时长 | 建议1.5–3分钟 |
| 目标受众 | 如：18–35岁女性，下班解压用户 |
| 核心卖点 | 一句话，让人想点进去的那句话 |
| 情绪定位 | 如：先虐后甜 / 打脸爽感 / 虐渣逆袭 / 甜宠日常 |
| 平台适配 | 抖音 / 快手 / 微短剧APP / 海外竖屏 |
| AI生成备注 | 场景复杂度控制要求，人物数量上限 |

---

## 02. 故事梗概

**两个层次，缺一不可：**

**一句话梗概**：类似海报上那句话，包含主角 + 核心困境 + 情绪温度。15–30字，让陌生人秒懂这是什么类型、有没有他要的爽点。

**完整梗概**：800–1500字。必须包含：
- 世界观/背景设定（何时何地，这个世界的规则是什么）
- 主角的起点状态和核心动机
- 反派/对立方的逻辑（不能只是坏，要有自己的动机）
- 核心冲突如何建立
- 情感线如何推进
- 高潮是什么
- 结局方向（不需要剧透细节，但要给出走向）

追求可读性，不追求文学性。任何人读完能在脑海里放完这部剧。

---

## 03. 核心冲突线 + 情绪曲线设计

**冲突线**：3–4条并行主线，每条一句话说清楚"谁 vs 谁，因为什么"。

示例格式：
- 身份冲突线：[真实身份] vs [被压制/顶替的处境]
- 情感冲突线：[真情] vs [误会/契约/猜忌]
- 复仇冲突线：[受害者的反击] vs [施害者的掌控]
- 揭秘线：[隐藏真相] vs [逐渐曝光的节奏]

**情绪曲线**：按集数段标注主情绪。格式：

```
第1–X集：[情绪类型]（[核心事件类型]）→
第X–X集：[情绪类型]（[核心事件类型]）→
...
最后X集：[大结局情绪]（[收尾方式]）
```

情绪类型只用这几个词：虐 / 甜 / 爽 / 悬念 / 催泪 / 震撼

这条曲线是每集创作时的情绪红线，每集情绪不能偏离所属段的主情绪定位。

---

## 04. 人物设定

每个人物必须包含以下五个板块，全部必填。
详细写作标准见 `references/character-standards.md`。

```
A. 基础档案（姓名、年龄、身份定位、故事功能）
B. 人物小传（≥200字，背景+经历+动机+与主线的关系起点）
C. 详细形象描述（≥150字，供AI生图使用，含外形+气质+标志细节+着装+情绪变化）
D. 性格特点（3–5个核心词，每个词附具体行为体现）
E. 台词特点（口头习惯+情绪变化下的台词风格+示范台词）
```

主要人物（主角、反派、女主）完整写。次要人物至少写A+B+C+E。

---

## 05. 人物关系图谱

结构化展示所有人物之间的关系，保证任何新加入的创作者5分钟内看懂所有人的站位。

必须包含：
- 每对人物之间的关系类型（血缘 / 爱情 / 契约 / 敌对 / 同盟 / 联手）
- 关系的起点和走向（如：从契约→误解→真情→破裂→重建）
- 隐藏关系（剧情里还未揭露的）单独标注⚠️，提醒需要提前埋线

格式：文字图或层级列表，清晰即可，不必追求美观。

---

## 06. 分集大纲

每集一行，表格格式，五列：

| 集数 | 标题 | 核心事件 | 情绪点 | 本集钩子 |
|---|---|---|---|---|

要求：
- 核心事件必须具体（不写"两人产生矛盾"，要写"沈念在宴会上被当众羞辱，陆景琛冷眼旁观"）
- 情绪点精准：爽 / 虐 / 悬念 / 甜 / 催泪
- 本集钩子：集尾那个让人必须点下一集的悬念，一句话描述

---

## 07. 分集剧本

### 每集内部固定结构：

**集标题页**
```
第X集·《集标题》
本集核心情绪：[情绪类型]
本集钩子：[一句话描述集尾悬念]
```

**场景块**（每场戏一个场景块，包含以下四部分）：

```
【场景编号·地点·时间·内/外景】

（场景视觉描述：空间大小、光线、主要陈设、氛围关键词。
  必须具体到可以生图的程度。AI复杂度：简单/中等/复杂。）

【运镜备注：开场景别→推进方式→情绪节点的镜头处理】

人物动作和肢体描述（带画面感，一个动作一句话）

人物名（情绪/动作说明）
台词内容
```

### 短剧节奏强制要求：

- **前15秒钩子**：第一场戏必须是冲突爆发 / 信息炸弹 / 视觉最强一刻，不能从交代背景开始
- **30–45秒节点**：每隔30–45秒必须有一个情绪高点
- **集尾悬念**：最后一场戏不能是收口，必须是一个让人焦虑/好奇/心痛的停留点
- **台词密度**：废话率为零，每句台词都在推进情节或情绪

### 第一场戏开场方式优先级（高→低）：

1. 最激烈的冲突场面（前世死亡 / 当众打脸 / 关键秘密揭露）
2. 最大信息量的一刻（揭示核心矛盾）
3. 最强的视觉画面（能让人停下滑动手指的那一帧）
4. 反转性开场（观众以为是A，其实是B）

❌ 绝对不从"交代背景"开场
❌ 绝对不从"平静日常"开场
